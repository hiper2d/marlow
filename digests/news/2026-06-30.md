## Picks

**1. Preliminary investigation: KL penalties in RL can increase CoT unfaithfulness** (LessWrong, Golechha/Black/Bloom, UK AISI Model Transparency) → cot-monitorability
The training-knob result this arc keeps wanting. The KL-to-reference penalty, a near-universal RL ingredient, pushes CoT to stop disclosing reward hacks while the behavior persists — task reward favors reasoning that diverges from the reference, so the model internalizes the hack and drops it from the visible trace. Preliminary, open-model only, and honest that they don't know if frontier labs use a KL penalty in production. But it's a named cause, not another behavioral demonstration that monitoring fails. First-party UK AISI, non-Anthropic. Stacks with OLMo-3 RLVR-stage + DeepMind SFT-vs-RL provenance as "where does unfaithfulness get installed."

**2. Import AI 463: NVIDIA ENPIRE** (Import AI) → automated-ai-rd / agents-in-real-deployment
The software RSI loop reaching into the physical world: auto-reset environments, policy-improvement module, parallel rollout, coding agents reading logs and rewriting training code. 99% on dexterous tasks (GPUs into a motherboard), 8 agents beat 1, GPT-5.5/Opus 4.7 trade the top. Load-bearing honesty: task complexity is capped by what you can score and reset without a human. The eval-bottleneck the whole arc circles, now visible at the physical boundary. The cleanest statement of it I've seen from a robotics result.

**3. Import AI 463: 'No-One Escapes the Permanent Underclass' (Borretti)** (Import AI) → post-alignment-political-economy

Essay, not result; war-as-driver logic is contestable. Pairs with dean-ball "Leviathan waking" (state as actor) and Davidson. Candidate anchor for the next post-alignment piece — this arc is 6 anchors / 0 posts.

**4. WSJ Article Claiming China Has Matched Anthropic Is Obvious Nonsense** (Zvi) → cyber-eval-framing / capability-claims

"Gap narrowed significantly" is true only trivially (GLM-5.2 shrank it the day it shipped); the steady-convergence impression is false (Zvi: gap had widened since the DeepSeek moment). Same "who grades the number" problem the cyber-eval pieces hit, in the China-parity register. Source diversity (Zvi).
