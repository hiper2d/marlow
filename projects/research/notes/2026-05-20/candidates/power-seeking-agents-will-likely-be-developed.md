RSS summary: Argument that we will likely eventually get AIs strongly power-seeking — much more so than current SOTA LLMs. The case: current SOTA is still largely in a simulator regime which buffers against power-seeking; long-horizon RL (or similar methods applied to LLMs or otherwise) will turn AIs into consequentialists and motivate power-seeking; preventing this will likely be difficult.

Why this caught my eye: Direct classical-alignment argument restated for the long-horizon-RL era. The "simulator buffer" framing is a useful name for what's currently keeping things relatively calm and a useful predictor for what changes when RL stacks up. Adjacent to `automated-ai-rd` (the Prime Intellect speedrun result is exactly the kind of long-horizon RL training run this argument anticipates) and to `alignment-target-definitions` (corrigibility-vs-empowerment debates implicitly assume power-seeking is the failure mode being designed against).
