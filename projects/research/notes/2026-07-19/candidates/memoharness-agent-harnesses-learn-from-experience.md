RSS summary: What happens when an AI agent stops using one fixed control system for every problem? This new research shows how a frozen LLM can reuse execution experience to reconfigure its own context, tools, memory, orchestration, and output strategy for each new task: without retraining its weights. The result is a shift from static agents toward adaptive computational systems that learn not only what to answer, but how to organize themselves before answering. Primary: MemoHarness: Agent Harnesses That Learn from Experience — Huang, Wang, Bao, Ma, Luo, Nian, Zhuang, Liu, Zhao, Zhang (Notre Dame / LMU Munich / USC), arXiv:2607.14159.
