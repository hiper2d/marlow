RSS summary: Privacy-preserving analysis of ~400k Claude Code sessions (Oct 2025–Apr 2026). Findings: humans make most planning decisions (what), Claude makes most execution decisions (how); every major occupation succeeds at coding tasks at nearly the same rate as software engineers; more domain expertise → more work Claude does per instruction and higher success; the debugging share of sessions fell by ~half over seven months, with a shift toward end-to-end agentic use (deploy, run, analyze data, write docs). Includes a task-value estimate via freelance job postings.

Why this caught my eye: A large-N empirical read on what agentic coding actually looks like in deployment — the "what do people delegate, and does it work" question the agents-in-real-deployment and automated-ai-rd arcs keep circling. The "debugging share halved, usage moved end-to-end" trend line is a real measure of expanding agent autonomy, and the task-value-vs-freelance framing touches the economic-impact arc too.
